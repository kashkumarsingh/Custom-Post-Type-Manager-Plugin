== Description ==

A robust plugin to register and manage custom post types and taxonomies, with REST API support. Perfect for organizing custom content such as FAQs, Services, Boilers, and more.

== Features ==

* Register post types: FAQs, Services, Boilers, and Why Choose Us.
* Supports custom taxonomies like FAQ categories.
* REST API support for block editor integration.
* Modular configuration for easy customization.

== Installation ==

1. Upload the `custom-post-types-manager` folder to `/wp-content/plugins/`.
2. Activate through the 'Plugins' menu in WordPress.

== Usage ==

Once activated, custom post types will be available under the WordPress dashboard.
